១. ដូចម្តេចទៅដែលហៅថា Internet?

Internet គឺជាបណ្តាញនៃការតភ្ជាប់ទំនាក់ទំនងពី Node មួយទៅកាន់ Nodes មួយទៀតពាសពេញពិភពលោក។ នៅក្នុងការតភ្ជាប់ Network គេអាចបែងចែកជា​ ៣ប្រភេទធំៗ ដូចជា៖

  1. LAN(Local Area Network): បណ្តាញដែរតភ្ជាប់មានលក្ខណះជិតៗគ្នា​ ឬ អាចក្នុងបន្ទប់, អាគារតូចៗជាដើម។
  2. MAN(Metropolitan Area Network): បណ្តាញនៃការតភ្ជាប់មានលក្ខណះធំជាងអាចទំហំប៉ុនទីក្រុងធំមួយ។
  3. WAN(Wide Area Network): បណ្តាញនៃការតភ្ជាប់មានលក្ខខណះទ្រង់ទ្រាយធំអាចពេញពិភពលោកទាំងមូលតែម្តង។

២. ដូចម្តេចទៅដែលហៅថា Websites?

Website គឺជាបណ្តុំនៃគេហទំព័រដែរត្រូវបានគេយកទៅដាក់លើ Internet សម្រាប់ឲ្យគេអាចប្រើប្រាសពាសពេញពីភពលោកតាមរយៈ ISP (internet Service Provice) មានដូចជា Facebook, Gmail, Google, Youtube,.....។​ អ្នកអាចធ្វើការប្រើប្រាស់នូវ Websites បានតាមរយៈការប្រើប្រាស់នូវ URL Website ឬ Web Address។

៣. ដូចម្តេចទៅដែលហៅថា Hosting?

Hosting គឺជាកន្លែងផ្ទុក Website សម្រាប់អោយវាអាចដំណើរការនៅលើ Internet បាន។ Hosting អាចផ្ទុកនូវពត៌មានដែលដំណើរការ Website បានដូចជា Documents, Video, Sound, Images...។

គេហទំព័រដែលអាចទុកចិត្តបានសម្រាប់ការទិញ Hosting និង Domain តាម រយះ Internet។ ចំពោះការទិញ Hosting និង Domain អាចទិញបានតាមរយះ Visa Card៕

ខាងក្រោមគឺជារូបភាព Server ដែលផ្ទុកទិន្នន័យរាប់លានទិន្នន័យដែលអ្នកអាច ជឿទុកចិត្តបាន។

៤. ដូចម្តេចទៅដែរហៅថា Domain?

Domain គឺជាឈ្មោះ ឬ Address ឬ URL របស់ Website ដែលជាតំរូវការ យ៉ាងចាំបាច់បំផុតសម្រាប់ការ យកគេហទំព័រដែលអ្នកបានបង្កើតហើយនោះដាក់ចូលក្នុង Internet។

៥. អ្វីទៅជា HTML?

HTML ឈរលើពាក្យថា Hyper Text Markup Language គឺជាភាសារ សម្រាប់បង្កើតនូវ Website ជាទម្រង់ Web Static ដំណើរការខាងផ្នែក Client។ HTML គេអាចប្រើប្រាស់ វាសម្រាប់ដាក់ Text, Video, Sound, Images...។
